## 4. 목록  

무순서 목록 3가지 방법  
####  '* 목록이름  
####  '- 목록이름  
####  '+ 목록이름  
  
### 순서가 있는 목록  
#### '숫자. 목록이름

하위 항목을 둘 때마다 일정한 칸 수로 지정해 주어야한다.

## 5. 코드블록

- 코드블록표현은 두 가지 방법이 있다.  
(1) 첫번째 행에서 세 개의 역따움표를 입력하고 '프로그래밍 언어 이름'을 입력하고  
두 번째 행에는 '코드내용'을 적고 세 번째 행에서 다시 세 개의 따움표로 닫으면 된다.  
(2) 공백 4개 또는 탭 1개를 붙인다
  
> '''프로그래밍언어이름  
  코드내용  
 '''

```python	   
 	def func();   
 	print "Hello"   
 	print "Paik"   
 	pass   
```

## 7.인라인 요소

+ 인라인 오쇼는 본문안에 필요한 형식을 넣는 요소들이다.
(하이퍼링크, 강조, 코드, 이미지 등이 있다.)  
1. 하이퍼 링크
   - 문단 중간에 다음과 같은 형식으로 하이퍼링크로 삽입 할 수 있다.
      >  [링크텍스트](링크URL "설명문구)

      ex) [My GitHub](https://github.com/paikscheol "My GitHub")
   - 위키 문서의 링크 
      > '' [[문서의 이름]]형식으로 비어있는 위키문서를 생성하는 하이퍼 링크를 만들 수 있다.  
        보이는 하이퍼링크의 이름과 실제 작성할 위키 문서의 이름을 다르게 만들경우   
        ''[[링크에서 보이는 이름 | 위키문서2]] 형식으로 작성할 수 있다.  

       ex)   
           [[위키문서]]  
           [[텍스트가 다른 문서| 위키문서2]]
2. 강조
   - 강조하는 방법 글씨를 굵게 표시하는 방법과 기울여 표시하는 방법이 있다.
   - 글씨를 굵게하려면 ** 텍스트 **(공백없이) __ 텍스트 __(공백없이) 작성하면 된다. 
   - ex) **이것은 테스트입니다**,  __이것은 텍스트입니다__
   - 기울여 표히사려면 * 텍스트 * _ 텍스트 _ (공백없이) 작성하면된다.
   - *텍스트* _텍스트_
3. 인라인코드
   - 문단 중간에 'Code'를 넣을 수 있다.
   - ex) `var data = 10;`
   
4. 이미지 넣기 
   - 링크(하이퍼텍스트)문법 앞에 느낌표(!)만 붙이면 뒤에 입력한 인터넷 주소의 이미지를 표시한다
     > ![트와일라잇](http://github.com/unicorn.png "tw") 

## 8. 자동링크
- 주소를 별도의 링크텍스트에 연결하지 않고, 주소 그대로 보여주면서 링크를 만드는 것으로 <와 > 로 감싸면 된다.  
ex)  
<http://www.naver.com>  

## 9. 탈출문자
- 기호 그대로 표시하게 하고싶을 때 사용 
- 표시하고 싶은 문자 앞에 역슬래시(\\)를 붙이면된다.  
ex)  
 __\*\*Hello\*\*__  
__\_hello\___
